
# Mine Sweeper
A simple application inspired by the famous Minesweeper game, built with Symfony and Symfony UX Live components

## Run Locally

Clone the project

```bash
  git@github.com:AsgarDev/mine-sweeper.git
```

Launch the internal server

```bash
  cd mine-sweeper
  symfony serve -d
  ...
  [OK] Web server listening
  http://127.0.0.1:8000
```
